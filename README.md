# Gider Takip Uygulaması

Bu proje, kullanıcıların günlük harcamalarını takip etmelerini sağlayan bir Android uygulamasıdır.

## Özellikler
* **Harcama Ekleme:** Kullanıcılar yeni harcamalarını sisteme girebilir (`AddExpenseActivity`).
* **Listeleme:** Ana ekranda harcamalar listelenir (`MainActivity`).
* **Detay Görüntüleme:** Harcamaların detayları incelenebilir (`DetailActivity`).

## Kullanılan Teknolojiler
* **Dil:** Java
* **UI Bileşenleri:** RecyclerView, CardView, Material Design, CoordinatorLayout
* **Kütüphaneler:** AppCompat (v1.6.1), Google Material Components (v1.11.0)
